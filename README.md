# ProgressiveOverload

AWS-focused training tracker for experienced lifters.

## Planned features
- Custom training intervals
- Configurable training days per interval
- Workout logging
- Exercise progress graphs
- Stagnation detection
- Progressive overload recommendations

## Planned stack
- Frontend: Next.js
- Backend: NestJS
- Database: PostgreSQL
- Infrastructure: AWS CDK
- Deployment: AWS Amplify / ECS Fargate / RDS / Cognito
